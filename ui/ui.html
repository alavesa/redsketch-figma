RedSketch
Threat-model your Figma designs
How it works
1. Select a frame in your Figma design
2. RedSketch reads the node names, text content, and structure
3. This data is sent to Claude AI for STRIDE threat analysis
4. You get a threat model with pattern matches and compliance gaps
Data & privacy
• Your design's node names, text, and hierarchy are sent to Anthropic's API over HTTPS for analysis
• Anthropic does not train on API data or store it beyond the request
• Your API key is obfuscated locally and never logged or shared
• No data is sent to any other service
• You can clear your key anytime in Settings
Setup
Enter your Anthropic API key to get started. Get one at console.anthropic.com
